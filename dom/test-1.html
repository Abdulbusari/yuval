The Document Object Model (DOM) Represents the HTML document.

The Document Object Model (DOM) is the data representation of the object that comprise the structure and content of a document on the Web.

The Document Object Model (DOM) is a programming interface for web documents. It Represents the page so that programs can change the document structure, style and content. The DOM Represents the document as nodes and objects; that way, Javascript can interact with the page.
